Este es un juego por turnos donde se puede escoger un nivel y mientras se finalize un nivel puede pasar al siguiente,
tambien dentro de este juego tenemos personajes de tipo Guardian quienes tienen nombre, tipo, vida, puntos de ataque y puntos de defenza,
El principal problema dentro de estre juego es la creacion de distintas listas y arrays de struct typedef de tipo Guardian
ya que se necesitan guardar distintos datos como lo es la lista principal donde se encuentran todos los guardianes y el array 
de guardianes que se encuentran jugando.

Para la resolucion de este problema he ocupado arrays y listas de struct typedef de tipo Guardian como mencione antes, ademas
he de ocupar Nodos para poder encadenar las listas y punteros para acceder a todos los datos a travez de funciones.
para finalizar se ha liberado el espacio de memoria de las listas y arrays utilizados.
